# Patient Mini Frontend Skeleton

This repository contains the patient mini-program frontend skeleton only.

Included:
- `src/App.vue`
- `src/main.ts`
- `src/pages.json`
- `src/manifest.json`
- `src/uni.scss`
- `src/components/`
- `src/pages/`
- `src/static/tabbar/`

Excluded on purpose:
- `src/api/`
- `src/stores/`

The goal is to let a UI-focused model or designer refactor page structure, layout, and styling without changing backend API contracts or request parameters.
